在本章中，我们将探讨通常在 API 网关层实现的以下常见用例。

* 带有 Kong Consumers 的 API 密钥
* 使用 Redis 限制速率
* 请求转换器
* 使用 Cognito 进行 OpenID 连接
* Lambda 函数抽象
* 代理缓存
* 使用 AWS Bedrock 的人工智能网关

这些功能可通过使用 Kong 插件进行扩展。您可以在 [Plugin Hub](https://docs.konghq.com/hub/) 上找到所有 Kong 插件的完整列表。

现在您可以点击**Next**开始模块。
