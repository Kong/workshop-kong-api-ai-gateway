In order to implement this use case we are going to expose a sample application implementing a **Marketplace**.

Here are the steps to implement the use case:
* Use the [Kong Mocking](https://developer.konghq.com/plugins/mocking/) plugin to mock an OpenAPI spec (OAS) as Kong Gateway Services and Routes.
* Apply [AI MCP Proxy](https://developer.konghq.com/plugins/ai-mcp-proxy) plugin to convert and expose the Kong Services as MCP Tools.
